# NEVVERLAND — navigation experiments

A study in navigation as a living organism, descended from the 2007
Firstborn site. One data tree, one physics simulation, seven switchable
experiments — from a springy type constellation to a perspective corridor
where your path recedes behind you in real z.

**Live: https://nessim-higson.github.io/nevverland/**

## The experiments

| Mode | Idea |
| --- | --- |
| 01 Organic | Type cascade on a force simulation — threads visible, everything sways |
| 02 Orbs | The circle constellation variant |
| 03 Structure | The cascade with invisible connections + copy blocks riding the focus |
| 04 Imagery | Structure + stills in z-space behind the type |
| 05 Depth | Imagery as attention: click a work and its image arrives as the environment |
| 06 Dust | One conserved pool of particles IS the site; navigation is phase-change |
| 07 Focal | The corridor: centered focus, past levels recede in true perspective z; arriving at work condenses the nav into one word while the media takes the stage (filmstrip / deck / roamable field / collage — switchable) |

A live physics panel (bottom-right) exposes the feel: type scale,
repulsion, spring, damping, energy, idle drift, looseness, rhythm,
spacing, cursor gravity, click shockwave.

## Run

```sh
bun install
bun run dev   # → http://localhost:4189
```

Content lives in `src/data.js` (any depth works; nodes take optional
`copy`, `img`, and `media` fields). Imagery from the Uniqlock sessions;
films generated from the stills with ffmpeg.
